# xianyu-publish（闲鱼发布助手）

[English](README.md)

把一件闲置从照片变成一条已发布的[闲鱼](https://www.goofish.com)宝贝的轻量、本地优先工作流——之后可选跟踪状态与互动数据，直到闲鱼显示已售。平台状态无法验证最终成交价。

你只需提供照片、回答几个「是/否」问题；查价、保价方案、诚实文案、发布（经你确认）与线上页面逐行核对，都由它代办。

## 安装

```bash
npx skills add Bing-Bryan/skills-by-bing --skill xianyu-publish
```

支持 Claude Code、Codex、Cursor 以及所有兼容 [`npx skills`](https://skills.sh) 的 Agent。

## 功能

1. **看图问事实** — 分析商品照片和描述，最多问 5 个关键问题（`是 / 否 / 不确定`）。
2. **低 Token 同类比价** — 第一层在本地聚合约 100–200 条搜索结果，不把完整原始数据送进模型；第二层只深读 15–20 条高度相关的个人卖家商品，判断成色、套装、挂牌价、互动和可信度。
3. **保价方案** — 一次给出：挂牌价、预期成交区间、仅向卖家展示并确认的心理底价、试售窗口。绝不自动降价。
4. **诚实文案** — 按「人—货—况—证—价—交」结构写手机端易读的文案；个人口吻、不用商家话术、每处瑕疵对应照片。
5. **确认后代发并核对** — 经你授权后通过 OpenCLI 或隔离浏览器发布，逐行核对线上页面（闲鱼会吞换行）。
6. **可选自有商品跟踪** — 发布后默认只跟踪你自己的商品：状态巡检和每日简报共用一个监控 Session；每 6 小时巡检，每天只汇总一次浏览/想要增量。

## 安全模型

- **默认 dry-run** — 发布、编辑、改价、下架、删除都需明确授权；降价永远单独确认。
- **底价仅卖家可见** — 确认方案时向你展示一次，之后以 `0600` 权限存于 `.xianyu-publish/`；日常报告、买家和商品页都不展示，售出即删。
- **不编造事实** — 不虚构功能、维修、保修、完整度；瑕疵如实披露、不做隐藏。
- **不绕过规则** — 遇登录失效、验证码、风控即停止，把操作交还给你。

## 依赖

- Python 3.9+（内置脚本仅用标准库）
- 可选：[OpenCLI](https://github.com/jackwener/opencli)，用于结构化的闲鱼搜索/读取/写入——安装前会先征求你同意；没有它也能完成看图、定价、文案
- 一个 Agent 可驱动的浏览器，用于登录和视觉验证

## 示例指令

- 帮我把这台相机挂到闲鱼卖掉
- 家里有件闲置，帮我看图查价、写闲鱼标题和文案
- 这部手机在闲鱼能卖多少钱？
- 帮我把这条闲鱼宝贝改价或下架
- Sell my old laptop on Xianyu — price it and write the listing
- 帮我盯一下我闲鱼链接的浏览量，每天报一次

本 Skill 只服务个人卖家；不处理买入/竞拍、商家批量上架、其他平台、数据集规模抓取或绕过平台规则。为一件个人闲置进行有限的同类比价属于正常范围。

## 实际案例：富士 X-S10 双镜套装

把商品照片放进工作目录后，用户从下面这一条 Prompt 开始调用 `$xianyu-publish`。Skill 随后只对功能、维修、票据、镜片磨损、整套出售和邮寄做了少量事实确认，便完成调研、定价、文案、发布、上线核对和轻量监控。

```text
$xianyu-publish

目录文件夹里面是一个富士的 XS10 的机器，所有的信息基本上都在文件夹的图中呈现了，它有很多的图片，如果还需要其他的图片，可以找我要。

还有其他的信息，就是我的 X-S10，它配套 15-45 的变焦镜头，以及 XC35 的定焦人像镜头，总共买下来是 9390。同时，还有两颗镜头的 UV 镜，以及 XC35 的 CPL 偏振镜，三支镜片总价值 270。

除此以外，我还赠送一块原装电池以及两块绿巨能电池，价值接近 200 左右。我还附送一个绿色的相机皮套，一个手持手电筒用于拍照，以及镜头清理套组，还有一个湿度显示计，这个在我的图片上都有显示，还有一个吹灰尘的那个，那叫什么东西我还不太确定，你帮我明确一下。

就是我送这么多东西，然后你帮我看一下，当然还有一个包，还有一个皮质的松紧收束包。帮我看一下这些东西总共卖多少钱比较合适？

还有一个信息是，我这些东西都是在 23 年国庆节买的。这一点也要诚实地说上来，帮我整合闲鱼上这个品类大致的价格综合判断一下，卖多少钱比较合理？再编辑一段挂到闲鱼上的描述。

关于图片，我再补充一点。所有微信 Image 打头的是一些多角度细节图，全家福三张图是我的所有物品。细节磨损方面，三张图展现出了物品的一些轻微磨损。
```

最终结果：富士 X-S10 + XC15-45 + XC35 F2，2023 年国庆购入，总价 9390 元；挂牌 6880 元，预期成交区间 6300–6600 元，使用 9 张图片，试售 7 天。

市场数据来自闲鱼个人卖家的在售报价，不代表实际成交价。价格调研图是 Agent 生成的分析界面，不是闲鱼原生页面。

## 目录结构

| 路径 | 用途 |
| --- | --- |
| `SKILL.md` | 工作流、能力阶梯、操作规则 |
| `references/` | 事实清单、定价、文案、监控规范 |
