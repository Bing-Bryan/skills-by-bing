# Lightweight fact intake

Inspect photos first. Ask no more than five questions, only when the answer changes price, trust, legality, or whether publishing is safe. Offer `yes / no / unknown` where possible.

## Universal core

- Exact item, brand/model/variant, and ownership
- Whether normal use is confirmed
- Repair, disassembly, water, impact, or other material history
- Visible and hidden defects
- Included core parts and important accessories
- Purchase time, warranty, receipt/invoice, and original price only when relevant
- Bundle/split preference, shipping/pickup, and desired timing

## Dynamic category checks

Generate only the checks relevant to the detected category. Examples:

- Electronics: battery, screen, ports, controls, activation/account locks
- Cameras: shutter/use count, sensor, autofocus, glass mold/scratches/haze/dust
- Appliances: power, noise, leakage, consumables, installation constraints
- Clothing/bags: size, stains, odor, repairs, hardware, authenticity evidence
- Furniture: dimensions, structural damage, disassembly, pickup constraints

Do not maintain a large fixed category questionnaire. If the user answers `unknown`, omit the claim or state the uncertainty plainly.

## Evidence handling

- Match each material flaw to a photo when possible.
- Do not expose precise addresses or full serial numbers.
- Distinguish expired warranty from missing warranty.
- Never infer authenticity, safety, or function from appearance alone.
