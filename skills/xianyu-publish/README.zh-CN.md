# xianyu-publish（闲鱼发布助手）

[English](README.md)

把一件闲置从照片变成一条已发布的[闲鱼](https://www.goofish.com)宝贝的轻量、本地优先工作流——之后可选轻量跟踪运营，直到以不低于你心理底价的价格成交。

你只需提供照片、回答几个「是/否」问题；查价、保价方案、诚实文案、发布（经你确认）与线上页面逐行核对，都由它代办。

## 安装

```bash
npx skills add Bing-Bryan/skills-by-bing --skill xianyu-publish
```

支持 Claude Code、Codex、Cursor 以及所有兼容 [`npx skills`](https://skills.sh) 的 Agent。

## 功能

1. **看图问事实** — 分析商品照片和描述，最多问 5 个关键问题（`是 / 否 / 不确定`）。
2. **同类比价** — 通过 OpenCLI 自适应采样 40–60 条真实个人卖家在售价，去重并剔除商家、回收、租赁、钓鱼价。
3. **保价方案** — 一次给出：挂牌价、预期成交区间、仅存本地的心理底价、试售窗口。绝不自动降价。
4. **诚实文案** — 按「人—货—况—证—价—交」结构写手机端易读的文案；个人口吻、不用商家话术、每处瑕疵对应照片。
5. **确认后代发并核对** — 经你授权后通过 OpenCLI 或隔离浏览器发布，逐行核对线上页面（闲鱼会吞换行）。
6. **可选轻量跟踪** — 每日汇总浏览/想要增量，降价前先诊断，单变量实验至少观察 72 小时。

## 安全模型

- **默认 dry-run** — 发布、编辑、改价、下架、删除都需明确授权；降价永远单独确认。
- **底价只存本地** — 以 `0600` 权限存于 `.xianyu-publish/`，绝不出现在买家可见内容中，售出即删。
- **不编造事实** — 不虚构功能、维修、保修、完整度；瑕疵如实披露、不做隐藏。
- **不绕过规则** — 遇登录失效、验证码、风控即停止，把操作交还给你。

## 依赖

- Python 3.9+（内置脚本仅用标准库）
- 可选：[OpenCLI](https://github.com/jackwener/opencli)，用于结构化的闲鱼搜索/读取/写入——安装前会先征求你同意；没有它也能完成看图、定价、文案
- 一个 Agent 可驱动的浏览器，用于登录和视觉验证

## 示例指令

- 帮我把这台相机挂到闲鱼卖掉
- 家里有件闲置，帮我看图查价、写闲鱼标题和文案
- 这部手机在闲鱼能卖多少钱？
- 帮我把这条闲鱼宝贝改价或下架
- Sell my old laptop on Xianyu — price it and write the listing
- 帮我盯一下我闲鱼链接的浏览量，每天报一次

本 Skill 只服务个人卖家；不处理买入/竞拍、商家批量上架、其他平台、批量抓取或绕过平台规则。

## 目录结构
